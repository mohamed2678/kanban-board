# Kanban ToDo Dashboard

## Objective
Build a Kanban-style ToDo list dashboard with 4 columns (Backlog, In Progress, Review, Done) using Next.js, React Query, and Material UI.

## Features
- Display tasks across 4 columns
- Create new tasks
- Update tasks (Drag & Drop between columns)
- Delete tasks
- Search tasks by title
- Optimistic UI updates using React Query
- Fully responsive layout with Material UI

## Tech Stack
- **Frontend:** Next.js, TypeScript, React Query, Material UI
- **State Management:** React Query
- **Drag & Drop:** @hello-pangea/dnd
- **API:** Next.js API Routes (In-memory CRUD)
